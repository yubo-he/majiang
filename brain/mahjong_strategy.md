# 川麻换三张打牌策略指南

## 一、换三张阶段策略

### 1. 换牌原则
- **同花色优先**：尽量换同一花色的牌，便于后续做清一色
- **中张牌优先**：3-7 的牌价值高，适合换出凑顺子
- **避免孤张**：不要换完全孤立的牌，可能帮对手成型
- **观察对手**：根据换回的牌判断对手做牌方向

### 2. 换牌类型
| 类型 | 策略 |
|------|------|
| 换好牌 | 换 3 张有用但非核心的牌，保留潜力 |
| 换废牌 | 换完全无用的孤张，快速定缺 |
| 换危险牌 | 换对手可能需要的牌，干扰节奏 |

## 二、定缺策略

### 1. 定缺判断
- **数量最少**：优先缺数量最少的花色
- **质量最差**：缺少搭子、对子的花色
- **看换牌**：换回的牌可能改变定缺决策

### 2. 定缺公开时机
- 每家在自己打出**本局第一张牌**时，公开自己的缺门
- 缺门从"推断问题"变为"已知事实"
- 后续一切推断与风险评估应以该缺门为硬约束

### 3. 常见定缺后手牌结构
```
理想结构：一门 10+ 张，一门 3-4 张（准备打掉）
平均结构：两门各 6-7 张（需快速决策）
危险结构：三门都有 4-5 张（容易点炮）
```

## 三、血战到底特殊规则

### 1. 血战机制
- 有玩家胡牌后，该玩家停止参与出牌
- 其余未胡玩家继续对局
- 对局结束条件：
  - 3 家已经胡牌（剩 1 家未胡），或
  - 牌墙摸完流局（仍有 ≥2 家未胡）

### 2. 目标函数随"已胡人数"变化
| 已胡人数 | 策略倾向 |
|----------|----------|
| 0 人胡 | 收益更看重"成牌速度 + 做大潜力" |
| 1 人胡 | 剩余玩家压力上升，倾向更快胡离场 |
| 2 人胡 | 剩 2 人血拼，防守权重显著上升 |

### 3. 行为限制
- **不能吃**：不能用别人打出的牌组成顺子
- **可以碰、杠**
- **一炮多响**：同一张点炮牌可被多家同时胡

## 四、打牌阶段策略

### 1. 早期（1-4 巡）
- **先打孤张**：单张的牌优先处理
- **先打边张**：1、9 的牌相对安全
- **保留搭子**：23、34、45、56、67、78 等好搭子
- **观察牌河**：判断哪些花色是热门

### 2. 中期（5-8 巡）
- **跟打熟张**：牌河出现过的牌相对安全
- **保护手牌**：不要随意拆搭子
- **判断听牌**：根据对手出牌速度判断进度
- **卡张处理**：4、5、6 的中张谨慎打

### 3. 后期（9 巡以后）
- **安全第一**：宁可流局不点炮
- **盯紧牌河**：计算剩余牌张数
- **防守优先**：明显听牌时打绝对安全牌
- **判断胡牌**：能胡就胡，不要贪大

## 五、读牌技巧（牌桌推断）

### S-01 早打 2/8 ⇒ 大概率无 1/3/4（或 6/7/9）
- **触发条件**：前 3 巡打出 2 或 8，且该花色不是缺门
- **推断结论**：该玩家该花色的 1/3/4（或 6/7/9）需求降低
- **正确性**：高
- **例外**：手牌极顺（已成 123）多出一张 2

### S-02 极早打 5（中张）⇒ 牌型可能"已成型/在做大"
- **触发条件**：边张未充分抛弃前，直接打出 5
- **推断结论**：更像对子体系（对对胡/暗刻）或清一色推进中
- **正确性**：中高
- **口诀**：**"极早打 5，非妖即怪"**

### S-03 先打 9 再打 8 ⇒ 7 相对安全
- **触发条件**：同一花色短时间内出现"9→8"拆边顺序
- **推断结论**：该玩家对 7 的需求显著下降
- **正确性**：高
- **口诀**：**"先打 9，后打 8，7 绝对安全"**

### S-04 打熟不打生 ⇒ 相邻生张更危险
- **触发条件**：中后盘疑似听牌玩家打出高熟度牌
- **推断结论**：熟牌本身安全，但相邻生张（±1/±2）风险上升
- **正确性**：中
- **典型**：手里有 3-4-4，摸进 2 后打熟牌 4，听 1/4

### S-05 断头牌效应
- **触发条件**：某张牌已出现 3-4 张（含副露）
- **推断结论**：两侧牌的顺子潜力暴跌
- **正确性**：高
- **示例**：7 死光后，8/9 基本成废牌（除非凑对子）

### S-06 碰牌后的孤立区
- **触发条件**：某玩家碰了 X
- **推断结论**：X-1 和 X+1 的联络性大减
- **正确性**：中高
- **示例**：碰了 4 万，3 万和 5 万的顺子需求显著下降

### S-07 晚期弃 3/7（核心张）⇒ 已成型/已听
- **触发条件**：中后盘弃出 3 或 7，且非绝张
- **推断结论**：听牌概率上升，相邻牌（1/4 或 6/9）风险上升
- **正确性**：中高
- **口诀**：**"金三银七，晚期抛弃必听牌"**

### S-08 清一色雷达
- **触发条件**：清完缺门后，连续且坚定弃出某一门，连 4/5/6 都弃
- **推断结论**：正在做清一色，保留的那一门危险度上升
- **正确性**：高

### S-09 晚期拆对子 ⇒ 多面听牌强信号
- **触发条件**：中后盘连续打出同一张牌两次（拆对子）
- **推断结论**：听牌概率接近 100%，且可能多面听
- **正确性**：高
- **示例**：连打两张 6 条，从单吊变成听 2/5/8 三张

### S-10 非线性出牌（先中后边）⇒ 诱导缺张
- **触发条件**：先弃 4 再弃 2（常规是先边后中）
- **推断结论**：可能手握 2-4 等 3 的卡窿，伪装放弃引诱别人打 3
- **正确性**：中
- **应对**：将中间缺失张（3 或 7）的危险权重调高

### S-11 周边死亡推暗对
- **触发条件**：X-1 与 X+1 大量外露甚至绝张，而 X 长期不出现
- **推断结论**：X 更可能被某家当对子/刻子保留
- **正确性**：中高
- **示例**：4 万和 6 万都绝了，5 万一张不见 ⇒ 肯定有人拿了 5 万对子

## 六、做牌方向选择

### 1. 清一色
| 项目 | 说明 |
|------|------|
| 条件 | 一门花色 10 张以上 |
| 番数 | 2 番 |
| 优势 | 番数高 |
| 风险 | 容易被察觉，对手会防守 |
| 技巧 | 早期隐藏意图，中期快速听牌 |

### 2. 对对胡（碰碰胡/大对子）
| 项目 | 说明 |
|------|------|
| 条件 | 手中有 3-4 个对子 |
| 番数 | 1 番 |
| 优势 | 碰牌快，进度明显 |
| 风险 | 碰牌后暴露意图 |
| 技巧 | 早期多碰，加快节奏 |

### 3. 七对子
| 项目 | 说明 |
|------|------|
| 条件 | 7 个对子（必须门清，无碰无杠） |
| 番数 | 2 番 |
| 优势 | 番数高，隐蔽 |
| 风险 | 成型慢，容易被抢胡 |

### 4. 清一色变体
| 牌型 | 组合 | 番数 |
|------|------|------|
| 清对 | 清一色 + 大对子 | 3 番 |
| 清七对 | 清一色 + 七对 | 4 番 |
| 龙七对 | 七对 + 四张相同 | 3 番 |
| 清龙七对 | 清一色 + 龙七对 | 5 番（封顶） |

### 5. 平胡（屁胡）
| 项目 | 说明 |
|------|------|
| 条件 | 牌型较散，无明显方向 |
| 番数 | 0 番（基础分 5 分） |
| 优势 | 隐蔽，容易胡牌 |
| 风险 | 番数低 |
| 技巧 | 快速听牌，多面听优先 |

## 七、牌河分析与剩余牌计算

### 1. 剩余牌判断
| 牌河出现 | 剩余张数 | 策略含义 |
|----------|----------|----------|
| 3-4 张 | 0-1 张 | 接近绝张，成顺概率极低 |
| 2 张 | 2 张 | 关键张谨慎 |
| 1 张 | 3 张 | 还有机会 |
| 0 张 | 4 张 | 可能都在手里（暗对/暗刻） |

### 2. 危险牌判断
- **绝对安全**：牌河出现 3-4 次的牌
- **相对安全**：对手刚打过的牌、1/9 边张、对手缺门花色
- **危险牌**：生张、中张 3/7、牌河未出现的牌

### 3. 关键张价值
- **金三银七**：3 和 7 是联络性最强的牌，可搭 12/24/45/56 等
- **中心牌 5**：可搭 34/46/67/78，价值极高
- **边张 1/9**：只能搭 23/78，价值最低

## 八、防守策略

### 1. 安全牌优先级
```
1. 牌河出现 3-4 次的牌（绝对安全）
2. 对手刚打过的牌（相对安全）
3. 1、9 边张（相对安全）
4. 对手定缺的花色（安全）
5. 自己手中对子的牌（较安全）
```

### 2. 点炮判断
- 计算剩余牌数
- 观察对手听牌范围
- 判断是否值得冒险

### 3. 弃胡时机
- 对手明显大牌
- 自己手牌太差
- 剩余牌数不足
- 牌河危险信号明显

### 4. 防守口诀
- **打熟不打生**：中后盘优先打牌河出现过的牌
- **跟打不跟听**：对手听牌迹象明显时，跟打其上家打过的牌
- **宁小不点炮**：宁可胡小牌或流局，不点大炮

## 九、常见牌型处理

### 1. 两面搭子（23、34、45、56、67、78）
- **保留**：好搭子，容易成顺
- **听牌**：两面听，胡牌概率高
- **价值**：最高

### 2. 边张搭子（12、89）
- **早期**：可保留，等中间牌
- **中期**：考虑优化
- **后期**：拆掉，效率低
- **价值**：最低

### 3. 卡张搭子（13、24、35、46、57、68、79）
- **早期**：可保留
- **中期**：考虑优化
- **后期**：谨慎，卡张难进
- **价值**：中等

### 4. 对子
- **早期**：可碰可留
- **中期**：根据方向决定（做对对胡则碰，做顺子则拆）
- **后期**：做将或碰牌
- **价值**：取决于做牌方向

## 十、计分规则（1.0 定版）

### 1. 基础计分
- **基础分**：5 分（0 番平胡）
- **胡牌分**：5 × 2^(min(总番数, 5))
- **封顶**：5 番（最高 160 分）

### 2. 结算方式
| 胡牌类型 | 结算方式 |
|----------|----------|
| 自摸 | 其余每家支付胡牌分给胡家 |
| 点炮 | 点炮者支付胡牌分给胡家 |
| 一炮多响 | 点炮者分别向每位胡家支付 |
| 杠上花 | 自摸，其余每家支付（含杠分转移） |

### 3. 杠分（刮风下雨）
| 杠类型 | 结算 |
|--------|------|
| 明杠（直杠） | 点杠那一家支付 10 分 |
| 暗杠 | 每家各支付 10 分 |
| 补杠（加杠） | 每家各支付 5 分 |

### 4. 流局结算
- **查花猪**：流局时手里仍有缺门牌，判为未听牌
- **查大叫**：未听者赔已听者"最大可能胡牌分"
- **退杠**：未听者需要退回杠分所得

```
换三张，想清楚，同花优先中张补。
定缺门，看数量，质量差也要考虑。
早打 2，必无 3，先 9 后 8 7 安全。
极早打 5 非妖怪，晚期拆对要警惕。
金三银七核心张，无故抛弃必听牌。
清一色，雷达响，疯狂清花要防备。
打熟张，防退熟，相邻生张更危险。
断头牌，废两边，周边死亡推暗对。
```
